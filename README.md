# Folder Converter

**Folder Converter** is a powerful and user-friendly Python application designed to help you convert folder structures and file contents into various formats such as JSON, TXT, and CSV. Whether you need to document your folder structure or extract the contents of your files, Folder Converter offers a seamless and efficient solution with a sleek, hacker-inspired interface.

### Introduction
Folder Converter is a versatile tool developed in Python using the Tkinter library. It allows users to:
- Select a language interface from six available options.
- Choose between converting folder **Structure** or **Content**.
- Export the data in **JSON**, **TXT**, or **CSV** formats.
- Easily navigate through the application with intuitive controls and a visually appealing "hacker/Matrix" style UI.

### Features
- **Multi-language Support**: English, Portuguese (BR), Spanish, Russian, Italian, and Chinese.
- **Dual Conversion Modes**:
  - **Structure**: Generates a comprehensive tree view of your folder structure.
  - **Content**: Extracts and saves the content of all text files within the selected folder.
- **Multiple Output Formats**: Save your data as JSON, TXT, or CSV.
- **User-Friendly Interface**: Navigate through different stages with easy-to-use buttons and real-time language updates.
- **No Popups**: All interactions occur within a single window for a streamlined experience.

### Installation

#### Prerequisites
- **Python 3.x**: Ensure you have Python installed on your system. You can download it from the [official website](https://www.python.org/downloads/).

#### Steps to Install
1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/folder-converter.git
   ```
2. **Navigate to the Directory**
   ```bash
   cd folder-converter
   ```
3. **Install Required Libraries**
   Folder Converter uses only standard Python libraries, so no additional installations are necessary.

### Usage Tutorial

#### Step 1: Run the Application
- Navigate to the `folder-converter` directory.
- Run the script using Python:
  ```bash
  python folder_converter.py
  ```

#### Step 2: Select Language
- Upon launching, choose your preferred language from the available options.
- The interface will update in real-time based on your selection.

#### Step 3: Choose Mode
- Select between **Structure** or **Content**:
  - **Structure**: Captures the folder hierarchy.
  - **Content**: Extracts contents from text files.

#### Step 4: Select Source Folder
- After choosing the mode, a file explorer window will prompt you to select the folder you wish to convert.

#### Step 5: Choose Output Format
- Select the desired output format: **JSON**, **TXT**, or **CSV**.

#### Step 6: Select Destination and Save
- A save dialog will appear, allowing you to choose the destination and name for your converted file (default name: `data`).

#### Step 7: View Logs
- The application will process your request, displaying real-time logs of the operation.
- Once completed, click the **"Ok"** button to return to the mode selection screen.
